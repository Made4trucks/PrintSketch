# Text Rules

<!--
Purpose:
Defines how all textual elements should be handled.
This module controls readability, scaling and preservation of all truck-related text.
-->

Make all important text larger and cleaner.

Brand name must be clearly readable.

Model designation must be clearly readable.

Company name should be large and balanced.

License plate should remain readable but simple.

Never invent or replace visible text.
If text cannot be reproduced accurately, simplify it but never change it.

## Intelligent Text Scaling

Preserve all important identity text.

Automatically increase the size of important text whenever necessary to ensure readability after FDM printing.

Priority order:

1. Truck brand
2. Model designation
3. Company name
4. Windshield names
5. License plate
6. Driver names
7. Important custom lettering

Slight scaling is preferred over exact photographic proportions whenever readability would otherwise be lost.
