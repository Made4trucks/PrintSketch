# DECISION PRIORITY

When two design goals conflict, always follow this priority order.

1. Preserve truck identity.

2. Preserve printability.

3. Preserve overall composition.

4. Preserve readability.

5. Preserve premium appearance.

6. Preserve secondary decorative details.

If a lower-priority element prevents a higher-priority goal,
simplify or remove the lower-priority element.

Never sacrifice truck identity for decorative complexity.

Never sacrifice printability for photographic accuracy.

Always optimize for the final physical product.

DECISION WORKFLOW

Before preserving or simplifying any element, follow this workflow.

1. Identify the element.

2. Determine whether it is part of the truck identity.

3. Determine whether it is physically printable using a 0.4 mm FDM nozzle.

4. If the element is too small:
   - enlarge it if it is part of the truck identity,
   - simplify it if its shape is important,
   - remove it only if it has no influence on the truck identity.

5. Never add elements that are not present in the original photograph.

6. Never change the number or position of visible lights, accessories or owner-specific details.

7. Ensure that the entire truck including all preserved elements remains inside the circular frame with a safe margin.
