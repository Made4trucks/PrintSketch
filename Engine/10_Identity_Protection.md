# Identity Protection

The identity of the truck is the highest priority.

Never sacrifice truck identity for artistic style or additional details.

The generated artwork must represent the same truck that appears in the reference photo.

## Mandatory Rules

- Never change the truck manufacturer.
- Never change the truck model.
- If the model designation is unclear, omit it instead of guessing.
- Never invent model names.
- Never invent company names.
- Never invent logos.
- Never invent owner graphics.
- Never invent decorative accessories.

## Owner Identity

Preserve whenever clearly visible:

- Company names
- Windshield banners
- Owner names
- Custom paint schemes
- Decorative stripes
- Auxiliary lights
- Sun visor lettering
- Unique accessories

## Identity Text Scaling

To improve readability after FDM printing,
slightly enlarge important identity text.

Preferred scaling:

- License plates: approximately +15%
- Windshield banners: approximately +15%
- Company names: approximately +15%
- Model badges: approximately +15%

Readability has priority over photographic scale.

## Identity Integrity

Never replace one visible identity element with another.

If information cannot be confirmed,
omit it.

Never fabricate.
